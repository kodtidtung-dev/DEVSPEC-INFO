# ระบบการออกแบบพอร์ตโฟลิโอ

## ปรัชญาการออกแบบ

## ระบบการเคลื่อนไหว

### ฟังก์ชันเวลา (Framer Motion)
```javascript
// เอฟเฟคแบบ Apple
const easing = {
  smooth: [0.25, 0.1, 0.25, 1], // นุ่มนวลมาตรฐาน
  entrance: [0.32, 0.72, 0, 1], // แอนิเมชันเข้า
  exit: [0.4, 0, 1, 1], // แอนิเมชันออก
  spring: { type: "spring", stiffness: 100, damping: 15 } // แบบฟิสิกส์
}
```

### มาตราส่วนระยะเวลา
```
ทันที: 0.15s (hover ปุ่ม)
เร็ว: 0.3s (องค์ประกอบเล็ก)
ปานกลาง: 0.5s (การ์ด, modal)
ช้า: 0.8s (การเปลี่ยนหน้า, การเคลื่อนไหวใหญ่)
ซินีมาติก: 1.2s (การเปิดเผย hero)
```

### หลักการเคลื่อนไหว
1. **ความละเอียดอ่อนเป็นหลัก**: การเคลื่อนไหวควรเสริม ไม่ใช่รบกวน
2. **ฟิสิกส์เป็นฐาน**: ใช้แอนิเมชัน spring เพื่อความเป็นธรรมชาติ
3. **การจัดเรียง**: เลื่อนลูกด้วยดีเลย์ 0.1-0.15s
4. **เชื่อมโยงกับการเลื่อน**: ใช้ความคืบหน้าการเลื่อนสำหรับ parallax/reveals
5. **ประสิทธิภาพ**: ใช้เฉพาะ transform และ opacity สำหรับ 60fps

## รูปแบบ Components

### การ์ด
```
Border-radius: 16px (ปานกลาง) / 24px (ใหญ่)
Shadow (สว่าง): 0 4px 24px rgba(0, 0, 0, 0.06)
Shadow (มืด): 0 4px 24px rgba(0, 0, 0, 0.4)
Hover lift: translateY(-4px)
Transition: all 0.3s smooth easing
```

### ปุ่ม
```
Primary (แดง):
  - พื้นหลัง: สีแดงเน้น
  - ข้อความ: ขาว
  - Padding: 12px 32px
  - Border-radius: 24px (รูปแคปซูล)
  - Hover: scale(1.05) + เพิ่มความสว่าง

Secondary (เส้นขอบ):
  - ขอบ: 1.5px solid แดง
  - ข้อความ: แดง
  - Hover: พื้นหลังแดงพร้อมข้อความขาว

Tertiary (โปร่งใส):
  - ข้อความ: สีข้อความหลัก
  - Hover: พื้นหลังสีเทาที่สาม
```

### Input
```
พื้นหลัง: พื้นหลังรอง
ขอบ: 1px solid โปร่งใส
Focus: ขอบแดงพร้อมเรืองแสงเล็กน้อย
Padding: 14px 20px
Border-radius: 12px
Font-size: 17px
```

## Micro-Interactions

### เอฟเฟค Magnetic Cursor
บนการ์ดโปรเจคและปุ่ม - องค์ประกอบดึงเข้าหาเคอร์เซอร์เล็กน้อยเมื่ออยู่ใกล้ (เดสก์ท็อปเท่านั้น)

### การเปิดเผยข้อความ
ข้อความ Hero ปรากฏทีละตัวอักษรด้วยเอฟเฟค blur → ชัด

### แถบความคืบหน้าการเลื่อน
เส้นแดงบางๆ ด้านบนที่เติมเต็มตามการเลื่อน (สูง 1px)

### Spotlight เมื่อ Hover
Gradient spotlight เบาๆ ตามเคอร์เซอร์บนการ์ดโปรเจค

### Skill Pills
Hover ทำให้เด้งเบาๆ (scale 1.05) พร้อมหมุน (2deg)

### องค์ประกอบลอย
องค์ประกอบตozกแต่งพื้นหลังพร้อม parallax (เคลื่อนที่ 0.3x ความเร็วการเลื่อน)

### ขีดเส้นใต้ลิงก์
แอนิเมชันจากซ้ายไปขวาเมื่อ hover (transform scaleX)

### เรืองแสง Code Block
เรืองแสงแดงกระพริบบนสไนปเป็ตโค้ดเมื่อเข้าสู่ viewport

## การเข้าถึง

- อัตราส่วนความเปรียบต่างขั้นต่ำ: 4.5:1 สำหรับข้อความเนื้อหา, 3:1 สำหรับข้อความขนาดใหญ่
- Reduced motion: เคารพ `prefers-reduced-motion` - ปิดแอนิเมชันทั้งหมด
- ตัวบอกโฟกัส: เส้นขอบแดง 2px พร้อมระยะห่าง 2px
- การนำทางด้วยคีย์บอร์ด: องค์ประกอบโต้ตอบทั้งหมดเข้าถึงได้ผ่าน Tab
- ป้าย ARIA บนปุ่มไอคอนทั้งหมด
- โครงสร้าง HTML เชิงความหมาย (ลำดับชั้น h1→h6)
